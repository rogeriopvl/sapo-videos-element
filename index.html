API

Available attributes

  • videourl (mandatory) - The URL of the video to embed
  • container (optional) - Selector with the element where the embed will be inserted. If not provided, the embed will be appended after the sapo-videos element
  • width (optional)
  • height (optional)
  • hd (optional)
